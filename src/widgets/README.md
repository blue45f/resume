# widgets Layer

페이지 수준의 큰 조각. 여러 features/entities를 조합하여 하나의 응집된 UI 블록을 구성합니다.

## 무엇이 여기 속하는가

- 여러 features/entities를 수평으로 결합하는 조합 컴포넌트
- 예: `HomeHero` (배너 슬라이더 + CTA), `RecommendationPanel` (이력서 추천 + 유사 채용공고), `DashboardStats`

## 이 프로젝트의 예상 위젯

- `home-hero/`: 메인 히어로 배너 슬라이더
- `recommendation-panel/`: 관련 추천 (이력서 / 채용 / 스터디그룹)
- `dashboard-stats/`: 통계 대시보드 카드 그리드
- `activity-feed/`: 최근 활동 피드
